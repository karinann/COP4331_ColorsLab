# COP4331 Colors Lab
COLORS web application made for COP4331 (POOSD) using LAMP stack on DigitalOcean.

Remote Linux server running LAMP on Ubuntu Droplet on Digital Ocean. Domain purchased through GoDaddy.

# Tech Stack
- Remote Linux server running LAMP stack on DigitalOcean
- Apache web server 
- MySQL database for colors
- API using PHP files

# Setting up Database
- Connect to server IP and sql into database
- Add users, colors, and contacts to database
